為替設定

最終更新日: 2026年4月2日

BudgetHub の為替は、標準設定のままでも利用できます。 必要な場合だけ、拡張設定からカスタムAPIを保存してください。

標準利用

為替は拡張設定を保存していなくても利用できます。 保有追加時に USD/JPY のようなシンボルを入力すると、標準の価格取得処理で評価額を計算します。

カスタムAPIを使う場合

  1. 設定 → 利用設定 → 資産設定 → 拡張 → 為替 を開きます。
  2. endpoint URL と APIキー を入力します。
  3. 保存 をタップします。

現在の実装では、為替でカスタムAPIを使うには endpoint URL と APIキーの両方 が保存されている必要があります。

入力形式

為替シンボルは USD/JPY の形式で入力してください。数量は左側通貨の保有量として扱われます。

カスタムAPIのリクエスト形式は他の資産種別と同じです。詳しくは API仕様 をご覧ください。
