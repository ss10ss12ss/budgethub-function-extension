セットアップガイド

最終更新日: 2026年4月2日

BudgetHub の資産設定では、資産種別ごとにカスタムAPIを保存できます。 現在の実装では、外国株式・国内株式・仮想通貨・先物は拡張設定に保存した内容を利用し、為替は標準設定のままでも利用できます。

基本の流れ

  1. 設定 → 利用設定 → 資産設定 → 拡張 を開きます。
  2. 設定したい資産種別を選びます。
  3. endpoint URL と必要に応じて APIキー を入力し、保存 をタップします。
  4. 資産 画面から保有を追加し、名称・シンボル・保有数量を登録します。

資産種別ごとの設定

外国株式 / 国内株式 / 仮想通貨 / 先物

これらの種別は、拡張設定に保存した endpoint URL を利用して価格を取得します。 APIキーが必要なサービスでは APIキー も入力してください。

APIキー不要のサービスであれば、APIキー欄を空のまま保存できます。

為替

為替は標準設定のままでも利用できます。カスタムAPIを利用したい場合は、為替設定画面で endpoint URL と APIキー の両方を保存してください。

保有追加時のシンボルは USD/JPY のような形式で入力します。数量は左側通貨の保有量として扱われます。

保有追加時の入力

  • 名称: 任意です。空の場合でも登録できます。
  • シンボル: 利用するAPIで識別できる値を入力します。
  • 保有数量: 数量または口数を入力します。
